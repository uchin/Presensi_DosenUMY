Pertanyaan Umum (FAQ)

Update terakhir : Januari 2017 | Oleh bsi UMY

Berikut adalah pertanyaan-pertanyaan umum yang biasa ditanyakan terkait aplikasi presensi dosen.

Download

Anda Bisa mendapatkan Aplikasi ini melalui web kepegawaian.umy.ac.id

Login, Username, Password

Untuk login, pada bagian username isikan email umy dan pada bagian passwor gunakan password yang diberikan oleh admin.

Belum memiliki Username, Password

Anda bisa tanya di bagian Biro sistem informasi Ext.164.

Untuk siapa Aplikasi ini

Aplikasi ini diperuntukkan bagi dosen UMY baik tetap maupun tidak tetap.

Apakah hanya digunakan di lingkunngan UMY

Ya untuk presensi check-in dan check-out Anda mesti terhubung ke jaringan UMY, jika Anda berada di luar koneksi UMY Anda hanya bisa melihat riwayat presensi.

Apakah Memerlukan data Internet

Ya jika Anda berada di luar Jaringan Internet UMY.

Apakah terintegrasi dengan Presensi Web

Ya. Aplikasi ini sudah terintegrasi dengan aplikasi presensi web.

Apakah aplikasi ini sebagai ganti Presensi Web

Tidak. Anda masih bisa melakukan presensi menggunakan aplikasi web.

Panduan Penggunaan Aplikasi

Update terakhir : Januari 2017 | Oleh bsi UMY.

Panduan ini ditujukan untuk mempermudah menjalankan aplikasi presensi dosen bagi pengguna.

Download dan Pembaruan

Anda Bisa mendapatkan Aplikasi dan pembaruan melalui web kepegawaian.umy.ac.id

Menjalankan Aplikasi

Setelah Anda Menginstall Aplikasi, temukan icon aplikasi presensi dosen seperti gambar di atas, kemudian tap icon tersebut.

Halaman Login

Silahkan isi username dengan email umy dan password dengan password yang Anda sudah miliki, untuk isian username dan password tidak boleh kosong, stelahnya tap tombol LOGIN.

Pada Halaman ini juga Anda bisa mengakses text link seperti lupa password, ketentuan layanan, kebijakan privasi, web bsi dan info-info lainnya.

Halaman Utama

Jika Login berhasil Anda akan berada pada halaman utama, jika Anda berada pada jaringan UMY maka tombol Check-in dan check-Out akan ditampilkan

Jika anda menggunakan koneksi internet di luar UMY maka tombol check-in dan ceck-out disembunyikan.

Melakukan Check-In dan Ceck-Out

Anda hanya bisa melakukan Check-In dan Check-Out, jika terhubung pada jaringan UMY.

Check-In Untuk Check-in tap saja tombol warna merah dan untuk Check-Out tap tombol warna hijau.

Jika tombol Check-In dan Check-Out masih blm muncul, melalui menu context pilih Refresh

Akan ada notifikasi jika Anda sukses atau gagal dalam melakukan Check-In dan Check-Out.

Detail Presensi

Untuk akses halaman ini Anda bisa lakukan melalui halaman utama (tombol warna biru) dan juga melalui menu Context.

Detail presensi menampilkan riwayat presensi bulanan, yang pertama ditampilkan adalah data sesuai bulan dan tahun berjalan, Anda bisa memilih bulan dan tahun melalui menu dropdown.

Menu Context

Bagian pojok kanan atas pada halaman utama Anda bisa menampilkan menu Context.

  • FAQ

    Pada menu ini Anda bisa meilhat pertanyaan-pertanyaan umum tentang aplikasi.

  • Panduan Pengguna

    Untuk mempermudah penggunakan Aplikasi Anda bisa membaca panduannya di menu ini.

  • Detail Presensi

    Pilih menu ini untuk menampilkan riwayat presensi bulanan.

  • Tentang Aplikasi

    Ini adalah halaman About, di sana Anda bisa mengakses beberapa info tentang versi Aplikasi serta beberapa link.

  • Refresh

    Ada kalanya tombol check-in dan check-out tidak muncul, jika hal tersebut terjadi coba pilih menu ini.

  • Tutup Aplikasi

    Untuk menutup aplikasi atau me Minimize aplikasi.

  • Logout

    Keluar dari aplikasi, jika Anda memilih ya maka Anda perlu Login Kembali.

Akses Web BSI dan Web Portal UMY

Melalui Aplikasi ini Anda juga bisa mengakses web bsi.umy.ac.id dan umy.ac.id, anda bisa membukanya melalui halaman About.

Akses Beberapa halaman Pendukung

Beberapa halaman pendukung telah kami sertakan seperti halaman FAQ, Ketentuan layanan, Kebijakan privasi dan Panduan Pengguna.

Penutup

Semua informasi yang ada pada gambar di atas hanya sampel saja dan bukan data sebenarnya.

Jika masih ada pertanyaan yang belum terjawab mungkin Anda bisa Mengakses halaman FAQ atau hubungi admin di Ext.164.
